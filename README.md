# Top Cuvée Activate

Static wholesale activation page for new Top Cuvée trade accounts. Buyer signs the
wholesale agreement (DocuSeal) and sets up Direct Debit (GoCardless) in one flow.

## Files

- `activate.html` — the page itself, no build step
- `top-cuvee-logo.png` — header logo
- `netlify.toml` — Netlify config (publishes the repo root, no build command)

## Deploy

Hosted on Netlify via GitHub import. Pushing to `main` redeploys automatically.
